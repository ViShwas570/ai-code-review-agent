# AI Code Review Agent

An AI-powered code review application built using Streamlit and Google Gemini API.

## Features
- Analyze GitHub repositories
- Detect code issues
- Suggest improvements
- Severity & confidence scoring
- AI-generated code review

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- GitPython

## How it Works
1. User enters GitHub repository URL
2. Repository is cloned locally
3. Python files are extracted
4. Gemini AI analyzes the code
5. Review suggestions are displayed
